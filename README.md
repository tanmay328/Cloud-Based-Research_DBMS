Pragyanm:Cloud-Based Research Data Management Portal
Pragyanm is a full-stack, cloud-based platform designed to securely store, organize, track, and share research datasets and documentation. It replaces fragmented traditional storage methods to prevent data loss and version conflicts for researchers, faculty, and students.  

Key Features
Secure Cloud Storage: Centralized management for all research files and outputs.  
Advanced Search (>60% Discovery Improvement): Fast dataset discovery via metadata tags, file types, authors, and names.  
Version Control: Tracks dataset updates cleanly to prevent accidental overwrites.  
Role-Based Access Control (RBAC): Custom interfaces and restrictions for Admins, Researchers, and Guests.  
Centralized Dashboards: Dedicated operational views for user uploads and system-wide admin analytics.  

Architecture & Tech StackFrontend: React.js, HTML5, CSS3, JavaScript  
Backend: Node.js, Express  
Database: MongoDB (handles metadata, user logs, and project indexing)  
Cloud & Infrastructure: Hosted on AWS EC2 with Nginx configured as a reverse proxy for secure, scalable request routing.
